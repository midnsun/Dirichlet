Моя лабораторная по лабе дирихле.
Чтобы запустить ее, нужно 
1. Поменять все пути в файле gui.py по подстроке "path" на Ваш путь к файлам
2. Поменять все пути в файле main.cpp по подстроке "path" на Ваш путь к файлам
3. Скомпилировать код c++ в бинарный файл executable.exe (именно такое имя)
4. Можно работать

gui.py можно запустить и без этого, но тогда не будет производиться никаких вычислений
